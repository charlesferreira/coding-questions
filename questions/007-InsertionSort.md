# Insertion Sort

Write a function that takes in an array of integers and returns a sorted version
of that array. Use the Insertion Sort algorithm to sort the array.

## Examples

### Example 1

#### Input

```javascript
array = [8, 5, 2, 9, 5, 6, 3]
```
